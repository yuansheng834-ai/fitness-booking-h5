# 健身约课 H5

这是小程序迁移用的 H5 工程。  
微信小程序原代码保留在工作区根目录，不受影响。

## 本地运行

```bash
cd D:\01-codex\h5-app
npm install
npm run dev
```

## 环境变量

复制 `.env.example` 为 `.env`，填写：

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## 当前进度

1. 注册、登录、角色路由守卫已完成。
2. 会员端：
- 首页约课（教练切换、日期切换、半小时课表、连续两格预约）
- 我的页面（历史预约按状态筛选、课时流水按月份筛选）
3. 教练端：
- 课表总览
- 点击预约看详情
- 标记完成
- 取消预约并返还课时
4. 管理员端：
- 课表总览（按教练切换）
- 用户账号管理二级页
- 设为教练、加减课时、增加当天取消机会
